The output ignored whitespaces and separates all arguments by one character. So each word is separated by one space. Other words are also separated by one space. For example, if the command ran is./step2 Hello     World. The output would be . / s t e p 2 H e l l o W o r l d. If the buffer is 1, The for loop handles it correctly because count is the valid bytes returned from the read. Read will return one byte at a time and then the for loop will write one byte at a time. A bigger buffer will use fewer iterations because read will return more bytes and then more bytes will be written at a time.
